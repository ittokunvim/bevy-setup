# bevy-setup

ここにはゲームエンジンBevyのセットアップコードが保存されています。

## 遊び方

リポジトリをクローンしてから、`cargo run`を実行することで遊ぶことができます。

## 操作方法

決定: 左クリック

キーボード操作: `A, B, D, R`キー
